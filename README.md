# ForzaMilan
Analyze soccer game stats to help Milan scouting

## Webapp design
- tab Player selection
- tab Metrics
  - one table
- tab Graph
  - season 1 (filter by player name and metric)
  - season 2 (filter by player name and metric)
  - ...

## Package required
- require(shiny)
- require(shinythemes)
- require(dplyr)
- require(magrittr)
- require(DT)
- require(RColorBrewer)
- require(stringi)
